# 🚢 PortNetX: Full-Stack Port Management & Data Platform

A comprehensive data engineering project demonstrating an end-to-end pipeline for maritime logistics, from raw data generation to secure web visualization.

## 🌟 Project Overview
PortNetX solves modern port congestion issues by providing real-time visibility into vessel traffic, sustainability metrics, and document security.

## 🛠️ The Tech Stack
- **Data Engineering:** Python (Pandas) & SQLite for ETL/ELT pipelines.
- **Backend API:** Flask (Python) with CORS for cross-origin data serving.
- **Frontend:** HTML5 & Tailwind CSS for a responsive, corporate-grade dashboard.
- **Security:** Blockchain-inspired SHA-256 hashing for eDoc verification.

## 📊 Core Modules
1. **Traffic Hub:** Live monitoring of vessel arrival, type, and delay status.
2. **Green Metrics:** Sustainability tracking to monitor carbon footprint reductions.
3. **Blockchain Ledger:** Cryptographic verification of manifest integrity.

## 🚀 How to Run
1. Run `python data_generator.py` to create raw data.
2. Run `python ELT_pipeline.py` to process and load data into SQL.
3. Run `python app.py` to start the backend server.
4. Open `index.html` in your browser to view the dashboard.
